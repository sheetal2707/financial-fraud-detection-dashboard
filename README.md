# 💳 Financial Fraud Detection Dashboard

## 📌 Project Overview

The Financial Fraud Detection Dashboard is an interactive Power BI project designed to analyze financial transaction data and identify fraudulent activities. Using Power BI, DAX, Power Query, and Microsoft Excel, the dashboard transforms raw transaction data into meaningful business insights, enabling users to monitor fraud patterns, transaction behavior, and financial risk effectively.

This project demonstrates end-to-end data analysis, including data cleaning, transformation, modeling, KPI creation, and interactive dashboard design.


## Objectives

- Analyze financial transaction data.
- Detect fraudulent transactions.
- Monitor fraud trends over time.
- Compare fraud across different transaction types.
- Track fraud amount and fraud rate.
- Present insights through an interactive dashboard.



## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel


## Dashboard Features

- Total Transactions KPI
- Total Fraud Cases KPI
- Fraud Amount KPI
- Fraud Rate KPI
- Fraud Trend Analysis
- Fraud by Transaction Type
- Interactive Filters (Slicers)
- Clean and Professional Dashboard Design


## Dataset

The project uses a financial transaction dataset containing transaction details such as:

- Transaction Type
- Transaction Amount
- Sender & Receiver Information
- Account Balance
- Fraud Status
- Flagged Fraud Status


## Key Insights

- Identified fraudulent transaction patterns.
- Compared fraud across different transaction types.
- Calculated overall fraud rate.
- Analyzed fraud amount distribution.
- Tracked fraud trends using transaction steps.
